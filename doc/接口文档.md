# 工时管理平台接口文档

## /user 用户操作

### GET 用户名重名校验 /user/checkName 

> request 

``````javascript
{
  'userName': string //需要校验的用户名
}
``````

> response

``````javascript
{
  'status': string, //响应状态
  'result': number  //1 可用，0 已存在不可用
}
``````
